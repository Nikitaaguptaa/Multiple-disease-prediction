# Multiple Disease Prediction System

This is my Multiple Disease Prediction project. I made this to predict diseases using machine learning.

## What it does

- User can check:
  - Diabetes
  - Heart Disease
  - Parkinson’s Disease
- Takes user input and gives prediction
- Simple interface using Streamlit
- Helps in early detection of diseases

## Tech used

- Python
- Machine Learning (Scikit-learn)
- Streamlit
- Pandas, NumPy

## Model Training

- First, I trained machine learning models using Jupyter Notebook
- Used datasets for Diabetes, Heart Disease, and Parkinson’s
- Applied algorithms like classification models
- Saved trained models using `.sav` files
- These models are then used in the Streamlit app for prediction
